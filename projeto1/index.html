O que é Javascript?

O JavaScript é uma linguagem de programação interpretada, sendo a principal linguagem cliente-servidor em navegadores web. Ela é responsável pelo dinamismo de uma página, ou seja, promover a interação com o usuário, sendo executados do lado do cliente. É uma linguagem script com orientação a objetos dinâmica e com tipagem fraca.
